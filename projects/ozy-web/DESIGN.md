# DESIGN.md Template (Bunker Style)

> Copia este archivo en la raíz de tu proyecto frontend para que el agente siga el sistema de diseño del Bunker.

## 🎨 Color Palette
- **Primary (Cyan)**: `#00e5ff` (Glow effect)
- **Secondary (Purple)**: `#a855f7` (Deep purple)
- **Background**: `#07090f` (Dark blue-black)
- **Surface**: `#0d1d2e` (Elevated cards)
- **Success**: `#10b981` (Green)
- **Text Primary**: `#ffffff`
- **Text Secondary**: `#ffffff70`

## 🖋️ Typography
- **Font Family**: 'Inter', sans-serif (UI), 'Fira Code', monospace (Code).
- **Scale**:
  - H1: 48px / 1.2
  - H2: 32px / 1.3
  - Body: 16px / 1.6

## 📐 Spacing & Borders
- **Unit**: 4px base (8px, 16px, 24px, 32px scale).
- **Border Radius**: 8px (standard), 13px (icons/buttons).
- **Glassmorphism**: 
  - Backdrop Blur: 10px
  - Border: 1px solid rgba(255, 255, 255, 0.1)

## 🔘 Components
### Buttons
- **Primary**: Solid Cyan, Black text, 8px radius, hover glow.
- **Ghost**: Cyan border, transparent bg, Cyan text.

### Cards
- **Dark Glass**: `#0d1d2e` bg, 10% opacity, 8px radius, subtle border.
