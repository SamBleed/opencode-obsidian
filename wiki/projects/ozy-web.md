# OZY-WEB

**Type**: Project / Frontend
**Status**: Active / Populated
**Stack**: React 19, Tailwind 4, Axios, Vite
**Blueprint**: [[DESIGN-TEMPLATE]]

## 📝 Description
The visual control center for the Bunker. Implements [[Design-System-MD]] and provides a live view of all [[Agentic-Entities]].

## 🚀 Key Features
- **Bunker UI**: Glassmorphism and glows following [[Design-System-MD]].
- **Live Sync**: Real-time fetching of [[Agentic-Entities]] from Postgres.
- **React 19**: Using the latest concurrent features.

## 🛠️ Commands
```bash
cd projects/ozy-web
npm install
npm run dev
```
