# Claudesidian

Pre-configured Obsidian vault + PARA method plugin by **heyitsnoah**. One of the earliest LLM Wiki Pattern implementations.

---

## Key Features

- `/init-bootstrap` — interactive setup wizard
- Vault import — safely imports existing vault to `OLD_VAULT/`
- Research public work to personalize context
- Thinking Mode vs Writing Mode distinction
- PARA folder structure: 00_Inbox, 01_Projects, 02_Areas, 03_Resources, 04_Archive
- Optional: Gemini Vision, Firecrawl

## Commands

- `/thinking-partner` — companion for deliberation
- `/daily-review` — end-of-day reflection
- `/research-assistant` — web research
- `/weekly-synthesis` — weekly consolidation
- `/inbox-processor` — clear inbox

## Comparison with opencode-obsidian

| Feature | Claudesidian | opencode-obsidian |
|---------|--------------|------------------|
| PARA import | Yes | No |
| Goal cascade | No | Future (v1.5.0) |
| Multi-depth | No | Yes |
| Skills | 5 | 10 |
