# Awesome DESIGN.md

A project by [[VoltAgent]] that provides a curated collection of [[DESIGN-MD]] files. 

## Purpose
To enable AI agents to generate UI that matches existing high-quality design systems by providing them with structured markdown specifications.

## Integrations
- **Google Stitch**: The concept originates from Google Stitch's design system documents.
- **OpenCode**: Can be used by OpenCode agents (like me!) to mimic specific styles upon user request.

## Notable Resources
- [getdesign.md](https://getdesign.md): The official web interface for the collection.
- [Request a DESIGN.md](https://getdesign.md/request): Service to request specifications for specific sites.

## Catalog Summary
The repository contains over 69+ (and growing) design specifications covering diverse industries from hyper-minimalist AI tools to complex automotive interfaces.
