# Goose (AAIF / Linux Foundation)

**Type**: General-Purpose AI Agent / Infrastructure
**Category**: Multi-Agent / MCP Native
**Status**: Researching / Integration Reference
**Parent**: [[_index]]

## 📝 Overview

**Goose** is an open-source, locally-running AI agent hosted by the **Agentic AI Foundation (AAIF)** under the Linux Foundation. It is built in Rust and focuses on extensibility via the **Model Context Protocol (MCP)**.

## 🚀 Key Features

- **MCP First**: Connects to 70+ tools and data sources via open standards.
- **Local Sovereignty**: Designed to run on user hardware for maximum privacy.
- **Model Agnostic**: Supports 15+ LLM providers.
- **Versatile Interface**: Desktop App, CLI, and API/ACP Server.

## 🧠 Connection to opencode-obsidian

Goose represents the **Infrastructure Layer**. While our vault is the *knowledge*, Goose is the *engine* that can consume it.

### Patterns to Adopt:
1. **MCP Compatibility**: Ensuring our skills and vault structure follow MCP standards for cross-agent use.
2. **Local Engine Optimization**: Leveraging our [[ENV-STATUS]] to ensure agents like Goose can operate optimally on this Arch Linux setup.
3. **Tool-Abstraction**: Thinking of our scripts not as "files", but as "MCP Tools" that can be exposed to any compliant agent.
