# Million.co

Million.co is an organization focused on React performance and developer tools. They are the creators of **Million.js** (a fast virtual DOM) and **React Doctor**.

## Products
- [[react-doctor]] — Health diagnostics for React codebases.
- Million.js — A drop-in replacement for the React virtual DOM that speeds up components.
