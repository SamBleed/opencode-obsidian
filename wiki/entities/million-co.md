# Million.co

Million.co is an open-source powerhouse led by Aiden Bai, focused on pushing the boundaries of React performance and developer tooling. Their philosophy centers on **"invisible performance"** — making apps faster without requiring developers to change how they write code.

## 🧪 Core Products

### 1. Million.js
A high-performance virtual DOM replacement for React. It uses a **"block-based"** reconciliation algorithm that can be up to 70% faster than standard React for data-heavy components.
- **Pattern**: It analyzes components at compile-time to determine which parts are static and which are dynamic.

### 2. [[react-doctor]]
A diagnostic tool that bridges the gap between "writing code" and "writing *good* code". It doesn't just find bugs; it enforces a philosophy of React development.

## 🧠 Philosophy: The Million Way
- **Compile-time > Runtime**: Move as much work as possible to the build step.
- **Actionable Metrics**: A score (like the [[react-health-score]]) is better than a long list of warnings because it creates accountability.
- **Agent-First**: Recognizing that AI is now part of the dev loop, they build tools that specifically help AI write better code.

## 🌐 Impact
Million.co has become a reference for the "New Wave" of React tooling, prioritizing speed, minimalism, and developer automation.
