## 📝 Overview

**Hermes Agent** is an open-source framework by **Nous Research** designed for persistent, autonomous AI agents. Its core philosophy is overcoming "AI amnesia" through a closed-loop learning system where the agent codifies successful procedures into reusable skills.

## 🚀 Key Features

- **Closed-Loop Learning**: The agent evaluates its own performance and creates "Skills" (Markdown docs) to handle future similar tasks.
- **Persistent Memory**: Uses SQLite and local file systems to maintain long-term context of user preferences and environment.
- **Model Agnostic**: Works with OpenAI, Anthropic, or local models via Ollama.
- **Multi-Interface**: Supports Terminal, Telegram, Discord, and Slack.
