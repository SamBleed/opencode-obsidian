# Awesome DESIGN.md

Curated collection of [[DESIGN-MD]] files extracted from real-world high-quality websites. These files are designed to be read by AI agents to achieve "pixel-perfect" UI generation.

## Core Concept
A `DESIGN.md` file is a plain-text design system document. It defines the "how the project should look and feel" part of the build process, complementing `AGENTS.md`.

## Key Categories
- **AI & LLM Platforms**: Claude, Cohere, ElevenLabs, Ollama, OpenCode AI, Replicate, etc.
- **Dev Tools & IDEs**: Cursor, Expo, Raycast, Vercel, Warp.
- **Backend & DevOps**: ClickHouse, MongoDB, PostHog, Supabase.
- **Productivity & SaaS**: Cal.com, Linear, Notion, Resend.
- **Design & Creative**: Figma, Framer, Webflow.
- **Fintech & Crypto**: Stripe, Revolut, Binance.
- **Big Tech**: Apple, Nike, Meta, Starbucks.
- **Automotive**: Tesla, BMW, Ferrari, Lamborghini.

## File Contents
Each entry typically includes:
1. `DESIGN.md`: The specification for AI agents.
2. `preview.html`: Visual catalog of components and tokens.
