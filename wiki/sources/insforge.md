# InsForge

The all-in-one, open-source backend platform for agentic coding. Gives coding agents database, auth, storage, compute, hosting, and AI gateway to ship full-stack apps end-to-end.

- **Stars**: 9.9k
- **License**: Apache 2.0
- **Language**: TypeScript (91.9%), Shell, PLpgSQL
- **Latest Release**: v2.1.6 (May 2026)
- **Site**: [insforge.dev](https://insforge.dev)

## Key Takeaways

- Backend platform designed specifically for AI coding agents to operate, not humans
- Dual interface: MCP Server (self-hosted + cloud) and CLI + Skills (cloud only)
- Agents read backend context, config, and logs to write, verify, and debug code autonomously
- Self-hosted via Docker Compose or one-click deploy on Railway, Zeabur, Sealos
- Cloud-hosted at insforge.dev
- Monorepo with Turborepo, 3,936 commits, 41 releases

## Core Products

- **Authentication**: User management, auth, sessions
- **Database**: Postgres relational DB
- **Storage**: S3-compatible file storage
- **Model Gateway**: OpenAI-compatible API across multiple LLM providers
- **Edge Functions**: Serverless Deno on the edge
- **Compute** (private preview): Long-running container services
- **Site Deployment**: Build and deploy sites
