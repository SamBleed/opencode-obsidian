# affaan-m/everything-claude-code

## Definition
Sistema de optimización de rendimiento para agentes AI harness. Provee 16+ agents especializados, 65+ skills, 40+ comandos y configuraciones de seguridad para Claude Code, Codex, OpenCode, Cursor y más.

## Key Features
- **16+ Specialized Agents**: desde "Frontend Wizard" hasta "Reddit Community Ninja"
- **65+ Skills**: Patrones comprobados para diferentes tipos de tareas
- **40+ Commands**: Comandos optimizados para flujos de trabajo
- **Security Harness**: Configuraciones de seguridad integradas
- **Memory System**: Sistema de memoria para agentes persistentes
- **Research-First Development**: Metodología que prioriza investigación
