# Self-Hosted AI Agent Frameworks 2026

## Resumen
Comparativa de los principales frameworks de agentes AI auto-hospedables en 2026.
Cubre LangGraph, CrewAI, AutoGen y n8n como las opciones líderes.

## Puntos Clave
- **LangGraph** (14K⭐): mejor para producción con state graphs, requiere Python
- **CrewAI** (25K⭐): mejor para prototipos multi-agente basados en roles
- **AutoGen** (38K⭐): Microsoft, multi-agente conversacional, incluye Studio no-code
- **n8n** (50K⭐): visual, mejor self-hosted dashboard, no requiere Python

## Datos Relevantes para el Bunker
- Todos corren via Docker
- n8n es el que tiene el dashboard self-hosted más pulido
- MCP support: n8n tiene soporte nativo; CrewAI con A2A
- Licencias: LangGraph MIT, CrewAI MIT, AutoGen MIT, n8n Sustainable AGPLv3
