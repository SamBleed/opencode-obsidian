# Awesome Claude Design

> 68 ready-to-use DESIGN.md files that Claude Design expands into full UI scaffold

Colección de 68 archivos DESIGN.md listos para usar. Cada uno describe un brand visual language que Claude Design convierte en un design system completo.

---

## Key Takeaways

- **DESIGN.md**: Markdown que describe identidad visual en formato que IA puede leer y actuar
- **Claude Design** (claude.ai/design): Workspace de diseño de Anthropic
- **9 secciones** por archivo: Visual Theme, Color, Typography, Components, Layout, Depth, Do's/Don'ts, Responsive, Agent Prompt Guide
- 68 brands organizados en categorías

## How It Works

1. Pick un DESIGN.md de la colección
2. Subilo a Claude Design
3. Scaffold: colors, typography, components, UI kit completo

## Output

Claude produce:
- README.md con brand context
- colors_and_type.css (CSS variables)
- preview/ cards
- index.html + components (working UI kit)
- SKILL.md portable
