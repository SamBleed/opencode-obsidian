# BrowserMCP

> MCP server + Chrome extension que permite controlar el navegador con IA

---

## Key Takeaways

- **MCP Server**: Servidor que permite a AI apps controlar el navegador
- **6.4k stars**, 494 forks
- **Stack**: TypeScript 89.2%, JavaScript 10.8%
- **Licencia**: Apache-2.0

## Features

- ⚡ Fast: Automatización local, sin latencia de red
- 🔒 Private: browser activity se queda en el dispositivo
- 👤 Logged In: Usa el perfil existente del navegador
- 🥷 Stealth: Evita bot detection y CAPTCHAs

## How It Works

Basado en Playwright MCP server, pero usa el navegador real del usuario en lugar de crear nuevas instancias. Permite mantener sesiones logueadas y evitar bot detection.

## Usage

Compatible con:
- VS Code
- Claude
- Cursor
- Windsurf
