# Bunker OS

A local-first knowledge operating system for OpenCode. BM25 text retrieval, n8n automation, 430 tests, zero external LLM dependencies.

## Principles

- **Local-first**: Everything runs on your machine
- **LLM-agnostic**: OpenCode is the orchestrator, you choose the model
- **Zero extra deps**: BM25 is stdlib Python — no ollama, no embeddings
- **Knowledge compounds**: Every session enriches the next
