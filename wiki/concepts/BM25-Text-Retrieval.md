# BM25 Text Retrieval

Fast, zero-dependency text search over wiki pages. Pure Python, no embeddings, no ollama. OpenCode reads the results and synthesizes.

```bash
python3 scripts/retrieve.py "query" --top 5
python3 scripts/retrieve.py build
```
