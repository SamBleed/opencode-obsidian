# Agent Skills Overview

This repository contains 23 structured engineering skills that encode production-grade workflows for AI coding agents. Each skill defines a disciplined process for tasks such as planning, implementation, testing, review, and shipping. The skills are organized into categories: Define, Plan, Build, Verify, Review, and Ship, and include detailed workflows, anti-rationalization tables, and verification requirements.

Key points:

- Skills enforce best practices from Google's engineering culture (e.g., test pyramids, trunk-based development, security hardening).
- Each skill ends with clear evidence requirements to ensure quality gates are met.
- The repository provides meta-skill guidance for selecting the appropriate workflow based on the work being performed.
- Integration with various IDEs and CLIs allows automatic activation of relevant skills during development sessions.

The ingestion process has created source and entity pages, and this concept page captures the high-level overview of the repository's purpose and structure.
