# Handover Protocol

> [!warning] Session Continuity
> Use this protocol when ending a session or transitioning between agents.

## 1. The Summary
Every session MUST end with a summary that covers:
- **What was done**: Bullet points of progress.
- **Mental State**: What was the agent thinking? (Logic paths, assumptions).
- **Next Clear Step**: What should the next agent do first?

## 2. Location
Handover notes reside in `wiki/meta/handovers/` (linked below) or as a `> [!abstract] Handover` block in the current project note.
