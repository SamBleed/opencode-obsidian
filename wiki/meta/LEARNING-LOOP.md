# Learning Loop Protocol (Estilo Hermes)

> Protocolo para transformar conversaciones exitosas en procedimientos reusables (Skills). 

## 🔄 El Ciclo
1. **Detección**: Al final de una tarea, el agente revisa: "¿Resolvimos algo complejo o repetitivo?"
2. **Extracción**: El agente identifica los pasos exactos que funcionaron.
3. **Codificación**: Se crea una nueva nota en `wiki/concepts/` o un nuevo `skill/`.
4. **Validación**: El usuario confirma si el procedimiento es correcto.

## 📝 Criterios para un nuevo Skill
- Se ha repetido más de 2 veces en la sesión.
- Involucra más de 5 pasos técnicos.
- Requiere configuración específica del entorno (ver [[ENV-STATUS]]).
- Es una "lección aprendida" tras un error (bugfix pattern).
